This test is only useful on devices that supports multiple cameras (e.g. a front-facing camera in addition to a primary camera).

Click the button, allow microphone and camera access, and start to capture a video.

Test passes

  1. if there appears a video capture screen,
  2. if there is a way to change camera,
  3. if there is a way to disable the audio record, and
  4. if there is a way to stop the video capture.
